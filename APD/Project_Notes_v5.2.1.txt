Scope
- Unify card rhythm across major features using tokens only; no structure changes, no !important.
- Lock visual edges to 16px and header→selector gap to 7px; collapse H2 ascent/descenders where needed.

Global Rhythm & Tokens (CEMENTED)
- --space-m: 16px (global rhythm unit)
- --space-header-offset: 7px (header→selector gap)
- --card-border-width: 2px (default; used for inset compensation)
- --cap-header-top-adjust: 2px (header nudge; can be tuned to 1px or 3px if a browser renders a hair off)

Card Inset (CEMENTED)
- Pattern: .card-content-container { padding: calc(var(--space-m) - var(--card-border-width, 2px)); }
- Result: 14px physical + 2px border ⇒ 16px visual inset (top/left/right/bottom).

Header Pattern (CEMENTED)
- .card-header { line-height: 0.7; margin: 0; padding: 0; transform: translateY(calc(var(--cap-header-top-adjust) * -1)); }
- Purpose: collapse font ascent/descender air; tokenized micro-nudge locks 16px top gap.
- Header→content gap: .card-header + * { margin-top: var(--space-header-offset); } (override per card if needed).
